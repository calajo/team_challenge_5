# Hundir la Flota (Team Challenge)

Este proyecto es una implementación en Python del clásico juego **Hundir la Flota (Battleship)** para jugar **desde la terminal**.

Toda la lógica del juego se ejecuta desde el archivo **`main.py`**, que actúa como **punto de entrada** del programa.

---

## Requisitos

- Python 3.8 o superior
- No se necesitan librerías externas adicionales (solo librería estándar de Python)

---

## Ejecución

1. Clona o descarga este repositorio en tu ordenador.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta:

   ```bash
   python main.py
